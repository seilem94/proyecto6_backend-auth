# Proyecto 6: Aplicación Backend con Autenticación

El objetivo de este proyecto es construir una aplicacion backend que admiistre la autenticación y autorizacion de los usuarios. Para eso se utilizará MongoDB de BBDD y Moongose como ODM (Object-Document Mapper) para la persistencia de datos, con un modelo de usuario y un modelo de "producto". Estos modelos estarán relacionados entre sí a través de MongoDB.

También se utilizará OpenAPI y Swagger para documentar la aplicación, y el código seguirá una estructura de carpetas clara con controladores, modelos, rutas y servicios. Finalmente, se desplegará la aplicación a través de render.com y MongoDB Atlas.
